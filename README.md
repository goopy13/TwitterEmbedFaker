# TwitterEmbedFaker
### [Invite](https://discord.com/api/oauth2/authorize?client_id=796836899293626388&permissions=117760&scope=bot)
Makes fake Twitter embeds trying to copy the ones Discord automatically generates when linking a tweet.<br/>
Uses [discord.py](https://github.com/Rapptz/discord.py) and [twython](https://github.com/ryanmcgrath/twython). To run the bot yourself you need a Discord bot token and Twitter api keys.<br/>
